# AWS Lambda Seed

Coding AWS lambda with ESNext

## Usage

### Install dependencies

```
$ npm install
```

### Build

```
$ npm run build
```

### Publish

NOTE: You need to manually create a function on AWS Lambda with the same name for the first time, then using this command to update it.

```
$ npm run publish
```
